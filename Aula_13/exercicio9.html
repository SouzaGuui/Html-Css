O que é HTML semântico?

É o uso de elementos com significado, como <article> e <section>, que descrevem a função do conteúdo.

Por que usar <main>?

Para indicar a área principal do documento, melhorando acessibilidade para leitores de tela.

Qual a diferença entre <section> e <article>?

<article> é independente e pode ser reutilizado; <section> agrupa conteúdo temático dentro da página.

Como estilizar <summary>?

Use CSS para ajustar tipografia, cursor e estados de <details> aberto.

O que é <aside>?

É conteúdo complementar ao principal, como notas, curiosidades ou boxes laterais.
